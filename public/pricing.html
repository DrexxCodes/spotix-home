Spotix Custom Event Page

This is the pricing section that helps you understand how pricing for custom events work

What is a Custom Event Page?

When you create an event on Spotix, you automatically have an event page where details of your event and eventual ticket purchase is done. However, a custom event page is a dedicated web page to your event details. This contains FAQs, more detailed descriptions, pictures of events (event collection), merch and so much more!

Please note that for event collection, individual events will use the Spotix IWSS checkout page while for regular one-off events will use Spotix IWSS popup.

Spotix Custom event page creation costs NGN 15,000.
