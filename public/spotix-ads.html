How will it work?

Even though it's not ready we have an overview how we want the API to work. We may make changes in production of the API but this is the main idea.

1

Signup

So obviously, the first thing you'll do is to sign up and as you sign up, you'll automatically get an API key. For non-techies, think of the API key as your house address.
2

Create a Project

Well we want to believe that some developers might have more than one project they would want to serve ads on and for that reason, developers can create projects which the ads will be served on and upon creating the project, you'll get a project ID. Non-techies think of this as a room inside your house.
3

Integrate

JS CLIENT

The JS client will contain the API URL which would be a JS file. This will be combined with the API key and Project ID to authenticate requests to the service. The NPM library will contain the bundled API for the service.
